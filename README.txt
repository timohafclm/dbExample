Для запуска программы на своем ПК необходимо скачать проект.
Собрать его можно с помощью Maven, для этого:
- скачать архив bin с официального сайта maven.apache.org;
- добавить в переменную окружения PATH адрес директории (куда распаковали архив maven);
- запустить консоль и перейти к директории со скаченным проектом;
- скомпилировать код с помощью команды 'mvn compile';
- после успешной компиляции запустить программу командой 'mvn exec:java -Dexec.mainClass="com.company.database.Main"';
